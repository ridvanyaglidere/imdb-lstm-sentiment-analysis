# IMDB LSTM Sentiment Analysis

Bu proje, IMDB film yorumları veri seti üzerinde **LSTM tabanlı bir duygu analizi modeli** uygulamaktadır. 

## Özellikler
- 100k kelimeyi içeren veri seti
- LSTM tabanlı derin öğrenme modeli
- Dropout ve Early Stopping ile overfitting önleme
- Eğitim süreci ve grafiklerle görselleştirme

**Test Sonuçları:**  
- Test Loss: 0.407  
- Test Accuracy: %82.5
